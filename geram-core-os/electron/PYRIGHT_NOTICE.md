# Pyright notice

GERAM CORE OS uses `pyright` 1.1.411 as its optional local Python language
server. The exact npm package and integrity hash are fixed in
`electron/package-lock.json`; GERAM never downloads it while the application
is running.

Project: https://github.com/microsoft/pyright
