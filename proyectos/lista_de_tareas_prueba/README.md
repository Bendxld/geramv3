# Aplicación Moderna de Lista de Tareas

Este es un proyecto de una aplicación web de lista de tareas ("Todo App") diseñada para ser moderna y funcional. Está construida utilizando tecnologías web estándar: HTML para la estructura, CSS para los estilos y JavaScript para la lógica interactiva.

## Características

*   Añade nuevas tareas a tu lista.
*   Marca tareas como completadas.
*   Elimina tareas que ya no necesitas.

## Cómo Usarla

Para correr esta aplicación, no necesitas instalar nada. Simplemente:

1.  Descarga o clona este repositorio.
2.  Abre el archivo `index.html` en tu navegador web preferido.
